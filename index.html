Monacoin Wallet

Monacoin Wallet app for your Android device. Standalone Monacoin node, no centralized backend required. Forked from the popular Bitcoin Wallet.

View project onGitHub

Welcome to Monacoin Wallet.

Monacoin Wallet for Android lets you keep Monacoins right with you on your phone. It is a port of the Bitcoin Wallet for Android by @schildbach.

About

The app is based on MonacoinJ, which in turn is a port of BitCoinJ (@bitcoinj). You can find MonacoinJ here: https://github.com/langerhans/monacoinj-new I ported all the needed changes for Monacoin compatibility over.

Download

You can get the app directly from the Google Play store: https://play.google.com/store/apps/details?id=de.langerhans.wallet

Alternatively you can download the current version for sideloading right here on this page.

Contributing

Please contact me before you want to contribute.

Building

Basicially you clone both this repo and MonacoinJ and compile it with Maven. More info is included in the README file.
